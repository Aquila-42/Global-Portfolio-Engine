# 📈 Quantitative Portfolio Optimization Suite

This repository is a collection of tools I built to explore the mathematical frameworks used in institutional finance. I wanted to see if I could build a robust system that bridges the gap between raw market data and strategic asset allocation.



## 💡 What’s Inside?

### ⚖️ Strategic Asset Allocation
The core engine uses `PyPortfolioOpt` to calculate the **Efficient Frontier**. It cleans raw `yfinance` data, handles multi-index headers, and provides discrete share counts for a given capital amount.

### 🌏 Cross-Border Analysis (US & India)
I designed a model to compare and combine high-growth US tech stocks with major Indian equities. It uses custom color-coding in Matplotlib to visualize how global diversification impacts the Sharpe Ratio.



### 🎲 Risk & Robustness (Black-Litterman & Monte Carlo)
To move beyond simple historical averages, I implemented:
* **Black-Litterman:** Injects subjective "market views" into the optimization to see how it shifts the weights.
* **Monte Carlo:** Simulates 5,000 random portfolios to create a "risk cloud," ensuring the optimal point isn't just a statistical fluke.

### 📊 Performance Showdown
An interactive **Plotly** dashboard that backtests the optimized portfolio against the S&P 500 and Nifty 500. It measures the "Alpha" (excess return) generated by the model over a 12-month out-of-sample period.

## 🛠️ Built With
* **Data:** yfinance, Pandas, NumPy
* **Finance:** PyPortfolioOpt
* **Viz:** Plotly, Matplotlib (Neon-Dark theme)

## 🏗️ The "Fun" Part
The biggest challenge (and the most fun) was writing the logic to handle transaction cost penalties and weight constraints—making sure no single stock takes up more than 15% of the portfolio to keep things realistic.
